# Pokemon App
Pokemon App est une application qui possedent plusieurs petites
fonctionnalités comme:
- un crud basique
- une page de details
- une page album qui présente de façon aléatoire des images
  de pokemon
- une page d'affichage et de filtrages des pokemons présent
